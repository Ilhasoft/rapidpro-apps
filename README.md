# RapidPro Apps

## Installing
```shell script
$ pip install -e git+git://github.com/Ilhasoft/rapidpro-apps.git
```

Add the desired apps to you INSTALLED_APPS setting:

**settings.py**
```python
...

INSTALLED_APPS += ("weni.<app_name>", ...)
```
